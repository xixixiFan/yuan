# 国际化

yuan-ui 组件库内置了国际化支持，可以根据不同的语言环境显示相应的文本。

## 支持的语言

| 语言 | 文件 |
| :---- | :---- |
| 简体中文 | zh-CN |
| 繁体中文 | zh-TW |
| 英语 | en-US |
| 日语 | ja-JP |

## 使用方法

在全局配置中设置语言：

```ts
import { createApp } from 'vue';
import YuanUI from 'yuan-ui';
import zhCN from 'yuan-ui/locale/zh-CN';

const app = createApp(App);

app.use(YuanUI, {
  locale: zhCN,
});
```

## 切换语言

在应用中动态切换语言：

```ts
import { createApp } from 'vue';
import YuanUI from 'yuan-ui';
import zhCN from 'yuan-ui/locale/zh-CN';
import enUS from 'yuan-ui/locale/en-US';

const app = createApp(App);

app.use(YuanUI, {
  locale: navigator.language === 'zh-CN' ? zhCN : enUS,
});
```

## 自定义文案

如果需要自定义特定组件的文案，可以在 locale 配置中覆盖：

```ts
import { createApp } from 'vue';
import YuanUI from 'yuan-ui';
import zhCN from 'yuan-ui/locale/zh-CN';

const app = createApp(App);

app.use(YuanUI, {
  locale: {
    ...zhCN,
    lang: {
      ...zhCN.lang,
      el: {
        pagination: {
          goto: '跳至',
          pageClassifier: '页',
        },
      },
    },
  },
});
```
